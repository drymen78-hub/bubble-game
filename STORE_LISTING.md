1. 난이도와 주제를 고르고 시작!
2. 물방울을 톡톡 터뜨려 그림을 또렷하게
3. 그림이 무슨 단어인지 맞히기
4. 모국어·학습어 발음을 듣고 따라 말하기
5. 15단어를 다 풀면 세트 완료! 또는 다른 주제로 도전

부모님과 아이가 함께, 또는 아이 혼자서도 안전하게 즐길 수 있는 단어 놀이입니다.

## 6. 데이터 안전 (Data Safety) 답변 가이드
- 데이터 수집: **수집 안 함 (No data collected)**
- 데이터 공유: **공유 안 함 (No data shared)**
- 저장 위치: 진도/설정은 **기기 내 localStorage에만 저장** (서버 전송 없음)
- 외부 연결: 폰트·발음 오디오 모두 **앱에 번들**, 제3자(광고/분석 SDK) **없음**
- 데이터 삭제 요청 방법: 앱 내 "진도 초기화" 버튼 또는 앱 삭제 시 즉시 제거
- 전송 중 암호화: 해당 없음(개인정보 전송 자체가 없음) / 앱 배포는 HTTPS
