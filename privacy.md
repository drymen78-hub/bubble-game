### 3. 기기 내 저장 데이터 (로컬 저장소)

앱은 게임 진행 상태를 **사용자의 기기 내부(LocalStorage)에만** 저장합니다.

저장되는 데이터:
- 점수(Score), 경험치(XP), 레벨(Level)
- 설정 화면에서 입력한 아이 이름 (선택 사항, 앱 내 표시용으로만 사용)
- 게임 설정 (소리 종류, BGM 여부 등)

이 데이터는:
- 외부 서버로 전송되지 않습니다.
- 제3자와 공유되지 않습니다.
- 앱 삭제 시 자동으로 삭제됩니다.
- 설정 화면의 "진도 초기화" 버튼으로 언제든지 직접 삭제할 수 있습니다.

Bubble Pop Picture Book ("the App") is a language learning game primarily designed for children under 13 years of age.  
The App treats children's privacy as its highest priority and **does not collect, store, transmit, or share any personal information.**

---

### 2. Information We Do NOT Collect

| Data Type | Collected |
|-----------|-----------|
| Name, age, gender, or other identifiers | ❌ Not collected |
| Email address or phone number | ❌ Not collected |
| Location / GPS data | ❌ Not collected |
| Device identifiers (Device ID, IDFA, GAID) | ❌ Not collected |
| Camera or microphone data | ❌ Not collected |
| Browsing history | ❌ Not collected |
| Advertising or tracking data | ❌ Not collected |
| Behavioral analytics | ❌ Not collected |

---

### 3. Data Stored Locally on Device

The App stores game progress **only on the user's device** (via LocalStorage):

- Score, XP, Level
- Child's name (optional, entered by user, used only for in-app display)
- Game settings (sound type, BGM, etc.)

This data is:
- Never transmitted to any external server.
- Never shared with third parties.
- Automatically deleted when the app is uninstalled.
- Deletable at any time via the **"🗑️ Reset Progress"** button in Settings.

---

### 4. Internet Usage

The App uses internet connectivity only for:

| Purpose | Personal Data | Description |
|---------|--------------|-------------|
| Font loading (Google Fonts) | ❌ None | Download display fonts |
| Text-to-Speech (TTS) | ❌ None | Uses device's built-in Web Speech API |
| PWA offline cache | ❌ None | Stores app files locally for offline use |

The App contains **no advertising networks, analytics SDKs, or social media SDKs.**

---

### 5. Children's Privacy — COPPA Compliance

This App complies with the **Children's Online Privacy Protection Act (COPPA)**.

- We do not collect personal information from children under 13.
- We do not sell or share children's data with third parties.
- The App contains no in-app advertisements.
- The App contains no in-app purchases.
- The App contains no external links or social features.

#### Parental Rights

Parents and guardians have the right to:

1. **Review** any data stored by the App for their child.  
   → Inspect via the browser's developer tools → Application → Local Storage.

2. **Delete** all stored data at any time.  
   → Use the **"🗑️ Reset Progress"** button in the App's Settings screen.  
   → Or clear browser/app data through device settings.

3. **Withdraw** from the App at any time without any data being transmitted upon doing so.

4. **Contact us** with any privacy-related questions or requests.

---

### 6. Security

All stored data resides on-device only. There is no external server, which eliminates the risk of server-side data breaches.  
Device-level security (screen lock, etc.) protects all locally stored data.

---

### 7. Changes to This Policy

We will notify users of any changes to this policy through app updates.  
For material changes, we will provide at least 30 days' notice prior to the change taking effect.

---

### 8. Contact

For privacy inquiries, data deletion requests, or any other questions:

- **Email:** drymen78@gmail.com
- **Response time:** Within 5 business days
