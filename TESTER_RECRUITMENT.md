# 비공개 테스트 테스터 모집 (품앗이) 가이드

> 목표: Play Console **비공개 테스트 20명 · 14일 연속 참여** 요건 충족 → 프로덕션 신청.
> 이 문서는 박철순이 그대로 복사해 쓰는 모집·안내 문구 모음입니다.

---

## 0. 먼저 알아둘 요건 (2024~ 신규 개인 개발자 정책)
- **테스터 20명** 이상이 **14일 연속** 비공개 테스트에 옵트인된 상태여야 함.
- 테스터는 **옵트인 링크로 참여**하고, 14일 동안 **앱을 설치·유지**해야 카운트됨.
- 그래서 "서로 깔아주고 14일 유지해주는" **품앗이**가 사실상 표준.

## 1. 품앗이를 구할 수 있는 곳
- 네이버 카페 검색: **"구글 플레이 비공개 테스트 품앗이"**, **"앱 테스터 품앗이"**
- 카카오 오픈채팅 검색: **"비공개 테스트 품앗이"**, **"클로즈드 테스트 20명"**
- 커뮤니티: 디스코드/Reddit `r/androiddev`(영어), OKKY, 인프런 커뮤니티 등
- 규칙(대부분 공통): **상대 앱도 똑같이 설치·14일 유지** 해주는 1:1 교환. 명단/스샷 인증 요구하는 곳 많음.
- 팁: 한 곳에서 20명 다 못 채우면 2~3곳 병행. 14일 카운터가 리셋되지 않게 **시작일을 맞춰 한꺼번에** 모으는 게 유리.

## 2. 사전 준비 (테스터 받기 전에 박철순이 끝내둘 것)
1. Play Console에서 **비공개 테스트 트랙 생성** + AAB 업로드(`twa-build/app-release-bundle.aab`).
2. 테스터 그룹(이메일 목록) 또는 **옵트인 URL** 발급.
3. assetlinks 지문 채우기 완료(앱 설치 시 주소창 안 뜨게) — `ROOTREPO_SETUP.md` 참고.
4. 아래 모집 글/안내문 준비(이 문서).

## 3. 품앗이 모집 글 템플릿 (카페·오픈톡에 게시)
```
[비공개 테스트 품앗이 구합니다 🙏 1:1 교환]

안녕하세요! 아이 단어 학습 게임 "버블팝 그림책" 비공개 테스트 함께해주실 분 찾습니다.
서로 깔아주고 14일 유지해드리는 1:1 품앗이로 진행해요. 상대 앱도 동일하게 테스트해드립니다!

📱 앱: 버블팝 그림책 (교육 · 유아 단어학습)
- 광고 없음 / 결제 없음 / 개인정보 수집 없음
- 버블 터뜨리며 그림으로 5개국어 단어 배우기
- 설치 후 한두 번 실행만 해주시면 됩니다(무리한 활동 X)

✅ 참여 방법
1) 참여 이메일(구글 계정) 알려주시면 테스터 등록해드려요
   (또는 옵트인 링크로 바로 참여)
2) 14일 동안 삭제하지 말고 유지만 부탁드려요
3) 제가 동일하게 상대 앱 설치·14일 유지로 보답합니다

댓글/쪽지 주세요! 빠르게 1:1 매칭해드리겠습니다 🙇
```

## 4. 테스터 참여 안내문 (매칭된 분께 보내기)
```
참여 감사합니다! 아래 순서로 부탁드려요 😊

1) 이 구글 계정으로 참여: (테스터 등록 완료된 이메일 확인)
2) 참여(옵트인) 링크 열기 → "테스터 되기" 버튼 누르기:
   👉 (Play Console에서 발급된 옵트인 URL 붙여넣기)
3) 같은 화면의 Play 스토어 링크로 "버블팝 그림책" 설치
4) 한 번 실행해 잘 켜지는지만 확인해주시면 충분합니다
5) 14일 동안 삭제하지 말고 유지 부탁드려요 (그 후 자유롭게 삭제 OK)

문제 있으면 편히 알려주세요. 상대 앱도 동일하게 설치·유지로 보답하겠습니다!
```

## 5. 14일 운영 체크리스트
- [ ] 비공개 테스트 트랙 생성 + AAB 업로드
- [ ] 옵트인 링크/테스터 그룹 발급
- [ ] assetlinks 지문 2개 채우기(주소창 제거) — `ROOTREPO_SETUP.md`
- [ ] 품앗이 모집 글 게시(2~3곳)
- [ ] 테스터 20명 이메일 수집 → Play Console 테스터 목록에 등록
- [ ] 시작일 기록(여기부터 14일 카운트): __________
- [ ] 14일 경과 + 20명 유지 확인 → **프로덕션 출시 신청**
- [ ] 내가 약속한 상대 앱들도 14일 유지(신뢰 = 다음 품앗이도 수월)

## 6. 막히기 쉬운 포인트
- **테스터 수가 "활성"으로 잡히려면** 설치+옵트인 상태 유지가 필요. 단순 이메일 등록만으론 부족.
- 14일은 **연속**이어야 함. 중간에 인원이 20명 아래로 떨어지면 카운터 영향.
- 앱 설치 시 **주소창(URL bar)이 뜨면** assetlinks 미설정 — 테스터 경험이 나빠짐. 출시 전 꼭 채울 것.
- 라이브 데모(웹)로 먼저 보여주면 모집이 수월: https://drymen78-hub.github.io/bubble-game/
