3. 기기 내 저장 데이터 (로컬 저장소)

앱은 게임 진행 상태를 사용자의 기기 내부(LocalStorage)에만 저장합니다. 저장되는 데이터:

  • 점수(Score), 경험치(XP), 레벨(Level)
  • 설정 화면에서 입력한 아이 이름 (선택 사항, 앱 내 표시용으로만 사용)
  • 게임 설정 (소리 종류, BGM 여부 등)

이 데이터는 외부 서버로 전송되지 않으며, 제3자와 공유되지 않고, 앱 삭제 시 자동으로 삭제됩니다. 설정 화면의 "진도 초기화" 버튼으로 언제든지 직접 삭제할 수 있습니다.

Bubble Pop Picture Book ("the App") is a language learning game primarily designed for children under 13 years of age. The App treats children's privacy as its highest priority and does not collect, transmit, or share any personal information. (Game progress is stored only on the device and is never transmitted — see Section 3.)

3. Data Stored Locally on Device

The App stores game progress only on the user's device (via LocalStorage): Score/XP/Level, the child's name (optional, used only for in-app display), and game settings. This data is never transmitted to any external server, never shared with third parties, automatically deleted on uninstall, and removable anytime via the "🗑️ Reset Progress" button in Settings.

  • We do not collect personal information from children under 13.

All stored data resides on-device only. There is no external server, eliminating server-side breach risk.
